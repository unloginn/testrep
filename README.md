# Git Practice Site

Простой HTML/CSS/JS сайт для тренировки Git и GitHub.

## Как открыть

Не открывай нихуя в пизде

## Идеи для практики

- Изменить заголовок сайта в отдельной ветке.
- Добавить новую секцию через Pull Request.
- Попросить второго участника сделать review.
- Специально изменить одну строку одновременно и решить конфликт.
- Создать Issue и закрыть его через PR с `Closes #1`.
